üéØ Usage Instructions

Step 1: Set up Telegram Bot

  1. Message @BotFather in Telegram
  2. Send /newbot command
  3. Follow instructions to create bot
  4. Copy the bot token
  5. Add bot to your target group/channel
  6. Get group ID using @userinfobot

Step 2: Configure Form Settings

  1. Open form builder in Q-Collector
  2. Go to form settings
  3. Enable Telegram notifications
  4. Enter bot token and group ID
  5. Test configuration

Step 3: Set up Field Notifications

  1. Select fields for Telegram notifications
  2. Set custom prefixes for each field
  3. Configure display order (1, 2, 3...)
  4. Preview message format
  5. Save field settings
